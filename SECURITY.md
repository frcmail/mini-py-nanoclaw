# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for potential vulnerabilities.

Report privately via GitHub Security Advisories:

1. Open the repository.
2. Go to `Security` -> `Advisories`.
3. Create a new private advisory with reproduction details and impact.

Include:

- Affected version/commit
- Reproduction steps
- Expected vs actual behavior
- Suggested mitigation (if available)

We will acknowledge receipt and provide remediation status in the advisory thread.
